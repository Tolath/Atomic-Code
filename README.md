# Atomic Code - Terminal Aktywacji Atomówki

A Fallout-style terminal game for activating nuclear weapons, built with React, TypeScript, and Tailwind CSS.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

The app will be available at http://localhost:3000

## Game Description

This is a puzzle game where you need to decode a cipher to enter the correct activation code. The game simulates a RobCo Industries terminal from the Fallout universe.

- A daily cipher key is provided
- Field officers report encrypted letters with codes
- You must decode the letters to form the target word
- Enter the digits in the order of the letters in the word

## Technologies Used

- React 19
- TypeScript
- Tailwind CSS
- Vite
- Lucide React icons
- Motion (Framer Motion)
